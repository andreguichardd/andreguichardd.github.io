## 🚀 Featured Projects

### 🏝️ [Tourist Segmentation — Tenerife](https://github.com/andreguichardd/tourist-segmentation-tenerife)
Unsupervised ML segmentation for the public tourism sector.
- K-Means clustering on **36,000+ real tourist surveys** across 123 behavioural features
- Produced **10 actionable visitor profiles** with statistically validated spending differences
- Kruskal-Wallis + Dunn-Holm hypothesis testing (p<0.05) — gaps of up to **€1,860/tourist**
- `Python` `Scikit-learn` `K-Means` `Statistical Testing` `Azure ML`
